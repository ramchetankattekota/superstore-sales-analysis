# 📊 Retail Sales Analysis Project (Superstore Dataset)

## 🧾 Overview
This project is a data analysis case study on a retail Superstore dataset.  
The goal is to analyze sales and profit trends using Python data science tools.

---

## 🎯 Objective
To explore and analyze sales data and extract meaningful business insights such as:
- Best performing product categories
- Region-wise profit analysis
- Top profitable products
- Sales trends

---

## 📂 Dataset
- Name: Sample Superstore Dataset  
- Records: 9994 rows  
- Columns: 21  
- Source: Kaggle Superstore dataset  

---

## 🛠 Tools & Technologies
- Python 🐍
- Pandas
- Matplotlib
- Seaborn

---

## 📊 Visualizations

The project includes the following graphs:

1. Sales by Category  
2. Profit by Region  
3. Profit by Category  
4. Top 10 Most Profitable Products  

---

## 📌 Key Insights
- Technology category has high profitability  
- Some regions perform better in profit generation  
- Certain products generate maximum profit  
- Sales distribution varies across categories
